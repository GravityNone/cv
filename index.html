Проекты

Умный дом с поддержкой Алисы

Создание веб-интерфейса и backend'а для управления устройствами умного дома с интеграцией Яндекс Алисы, ESP32, базы данных и Telegram-бота.

Система распознавания лиц

Программа на Python + OpenCV для Raspberry Pi, которая определяет лица в реальном времени и синхронизируется с системой «умного дома».

RFID-система присутствия

ESP32 + RFID + база данных. Система регистрации приходов/уходов с отображением в веб-интерфейсе и уведомлениями в Telegram.

Графики и отчёты по устройствам

Сбор и визуализация данных с устройств в виде графиков, адаптивная панель управления с аналитикой и фильтрацией по типам устройств.

Включение ПК через Wake-on-LAN

ESP32 и Raspberry Pi взаимодействуют с роутером и сетевыми адаптерами, чтобы включать ноутбук Asus через Алису и Flask-сервер.
